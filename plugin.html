Rebuilding the plugin

Google has their own branch of the unreal engine in 4.13. It currently contains updated versions of the Google VR plugin. To use the pointer, we must use that version. We are currently at version 4.14, and later 4.15. Instead of waiting hours to compile (depending on your computer) and trying to update the branch, we can just update the plugin.


1. Make sure you have Github linked with Unreal and download Google's branch of Unreal.
2. Create a new empty C++ project in Unreal.

3. Open the zip and go to Engine/Plugins/Runtime/GoogleVR

4. Copy the GoogleVRController folder into the Plugins folder of your project.
Note: If a Plugins folder does not exist, create it.

5. Open Visual Studio and build the project.
Note: If the Plugins folder is not showing in Visual Studio, you may need to Rescan Solution and/or restart VS/Unreal


Congrats! You have compiled the plugin!
